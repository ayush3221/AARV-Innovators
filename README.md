# SmartETP: Industrial SCADA & Compliance Simulator

An advanced Effluent Treatment Plant (ETP) monitoring system designed for the **Integrated Iron & Steel Industry**. 

## 🚀 Features
- **Real-time Telemetry:** Simulates 8+ chemical parameters (BOD, COD, Cyanide, etc.).
- **Chemical Engineering Logic:** Models based on Stokes' Law and First-Order Kinetics.
- **CPCB Compliance:** Automated PASS/FAIL auditing against Iron & Steel industry norms.
- **Exportable Logs:** Downloadable CSV/Excel audit reports with IST timestamps.

## 🛠️ Tech Stack
- Backend: Python (Flask)
- Database: SQLite
- Frontend: Chart.js (Gradient Analytics), Bootstrap 5

## 📥 Installation
1. `git clone https://github.com/ayush3221/SmartETP.git`
2. `pip install -r requirements.txt`
3. `python app.py`
